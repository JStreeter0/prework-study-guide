HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • HTML attribute: defines additional info about element
  • Metadata:Date about our webpage that the browser uses to serve webpage user, typically not seen
  • SEO(search engine optimization): Process of improving the ranking of a website in search engines
  • Elements nested inside elements are child elements, holding element is referred to as parent element ex. head or body
  • Semantic Element:The name of the element describes its content and the role it plays on a page
  • Element/Tag difference, element is from open to close, tag is what appears in angle bracket
  • h1 should only appear once per page
  • title:defines title for webpage
  • style:defines css styles associated with page
  • base:defines base url
  • link:connects external resources to html doc
  • meta:defines metadata-character set, description, keywords, author, and viewport
  • script:element embeds client-scripts in html client
  • h1-h6:headers, H1 being largest
  • p:paragraph block
  • ul,ol,li:unordered list, ordered list, list item
  • img:info about images displayed
  • src:Attribute defines location of the image
  • a:stands for anchor,creates links to same or other webpages
  • href:attribute points to url for link
  • br:line break
  • section:semantic element that defines a section in a doc that contains thematic content
